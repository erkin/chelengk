* Chelengk
  A work-in-progress project for a symbolic [[https://en.wikipedia.org/wiki/Turkish_makam][makam music]] engine based on
  Arel-Ezgi-Uzdilek system.

*** Example use
#+BEGIN_SRC racket
  > (get-makam 'rast)
  (rast dügah segah çargah neva hüseyni eviç gerdaniye)
#+END_SRC

** Roadmap
