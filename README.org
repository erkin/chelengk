* Chelengk
  A work-in-progress project for a symbolic [[https://en.wikipedia.org/wiki/Turkish_makam][makam music]] engine.

*** Example use
#+BEGIN_SRC racket
  > (get-makam 'rast)
  (rast dügah segah çargah neva hüseyni eviç gerdaniye)
#+END_SRC

** Roadmap
   + Compound makamlar
   + Makam developments
   + MIDI and MusicXML export
